# Cara menggunakan

## Clone this project
```git
git clone https://github.com/rifkinasss/spk-saw-laravel.git
```

## Buka di VS Code

## Gunakan Command Berikut
```
1. cp .env.example .env
2. php artisan key:generate
3. php artisan migrate --seed
4. php artisan serve
```
